1.What is dom? What is the purpose of it?

Ans: DOM is a Document Object Model and Programming interface for html .It use for connect html document with javascript. By using DOM we can get,change ,add and remove html element.We can add many interaction in html element with js by using DOM.

2.How to use DOM element?

Ans: Ans:There are many ways to use DOM. We can get element by tag name with the getElementByTagName(). We can also get element by uniq id or class with the getElementById() or getElementByClassName. We can also use querySelectorAll for get list of the documents element and querySelector for get first element of a list of the documents element .

3.What is event bubble?

Ans: Ans:Event bubble is easy to define with an example.Suppose there is a section in a html document and under the section an unordered list and there are event handlers in section,ul,li. When event happens on a list item , it first runs the handlers on it, then on its parent ul, then on its parent section.This overall things called event bubble.
